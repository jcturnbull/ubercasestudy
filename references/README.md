# References

External documentation and lookup data used by the project.

- `data_dictionary_trip_records_hvfhs.pdf`  
  NYC TLC data dictionary for HVFHV trip records.

- `isd-format-document.pdf`  
  NOAA ISD (global-hourly) data format specification.

- `taxi_zone_lookup.csv`  
  Taxi zone lookup table (zone IDs, borough, and zone names).

These files are not modified by the pipeline; they are used as reference when interpreting data fields and zones.
